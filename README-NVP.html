Java SDK for PayPal NVP API

About the PayPal Java SDK for NVP API

The PayPal's Software Developer Kit (SDK) eases the process of integrating PayPal's financial services into your application by providing a number of reusable libraries and tools. This software is intended for use by software developers with a basic understanding of the selected platform's language and environment.

The latest upgrades, FAQs and announcements can always be found at: Tools for PayPal Developers

Download
The PayPal Java SDK requires you to encrypt your API Certificate into PKCS12 format. This step requires the OpenSSL encryption tool. The Win32 OpenSSL Installation Project is included in the SDK distribution

Proxy Support

Proxy support has been added for environments behind an HTTP proxy.  To enable, enter proxy information into the sdkproxy.properties.  For details on the parameters, refer to comments in WEB-INF\classes\sdkproxy.properties located in the paypaljsp.war file.
